Este proyecto utiliza un Arquetipo Maven para asegurar que todos los microservicios compartan la misma configuración técnica (Java 21, Spring Boot 3.5).

Requisitos del Entorno
JDK: Versión 21.
Visual Studio con Extension Pack for Java.
Maven: No es necesario instalarlo (se incluye Maven Wrapper).

Comandos de Ejecución
Desde la raíz del proyecto, utiliza los siguientes comandos según sea necesario:

Ejecutar el Microservicio desde la terminal del proyecto:
  .\mvnw spring-boot:run

Ejecutar test del Microservicio desde la terminal del proyecto:
  .\mvnw test
